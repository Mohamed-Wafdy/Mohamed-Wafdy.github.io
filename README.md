# mohamed wafdy

This project was built using Vite, TypeScript, React, shadcn-ui, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```sh
# Step 1: Clone the repository
git clone <repository-url>

# Step 2: Navigate to the project directory
cd <project-directory>

# Step 3: Install dependencies
npm install
```

### Development

```sh
# Start the development server
npm run dev
```

### Build

```sh
# Build for production
npm run build
```
